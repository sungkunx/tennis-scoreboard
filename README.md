# 테니스 점수판 (Tennis Scoreboard)

모바일 최적화된 테니스 토너먼트 점수판 웹앱입니다.

## 🎾 주요 기능

### 📱 모바일 최적화
- 터치 친화적 인터페이스
- 반응형 디자인
- 모바일 웹앱으로 사용 가능

### 🏆 토너먼트 관리
- **모임 생성**: 날짜, 멤버 정보 입력
- **랜덤 생성**: 자동 멤버 생성 (AAA, BBB, CCC...)
- **멤버 관리**: 이름, 성별(남/여/없음), 실력(1-10) 설정

### ⚙️ 경기 설정
- **대전 방식**: 4가지 매칭 옵션
  - 실력/성별 완전 랜덤
  - 실력만 분리
  - 남녀만 분리
  - 실력/성별 완전 분리
- **게임 형식**: 4게임 또는 6게임
- **코트 설정**: 코트 수, 매치 수 조절

### 📊 실시간 스코어링
- **순위표**: 승수 → 게임득 → 게임실 순 자동 정렬
- **점수 입력**: 실시간 순위 업데이트
- **경기 스케줄**: 코트별 경기 일정 관리

### 🌐 공유 및 협업
- **서버 모드**: 코드 입력으로 여러 명이 데이터 공유
- **게스트 모드**: 로컬 전용 사용
- **공유 링크**: 경기 현황 실시간 공유
- **기존 모임 복사**: 멤버 정보 재사용

### 💾 데이터 관리
- **자동 저장**: 로컬스토리지 또는 서버 저장
- **결과 정리**: 최종 순위 및 통계 요약
- **상태 복원**: 중단된 경기 이어서 진행

## 🚀 사용 방법

1. **첫 접속**: 코드 입력 또는 게스트 모드 선택
2. **모임 생성**: 새 모임 또는 기존 모임 복사
3. **멤버 설정**: 참가자 정보 입력
4. **경기 설정**: 매칭 방식 및 게임 설정 선택
5. **점수 입력**: 실시간으로 경기 결과 입력
6. **결과 공유**: 공유 링크로 다른 사람들과 현황 공유

## 🛠️ 기술 스택

- **Frontend**: React (CDN), Tailwind CSS
- **Storage**: LocalStorage + 서버 연동 준비
- **Deployment**: Vercel/Netlify 호환

## 📱 모바일 사용 팁

- 홈 화면에 추가하여 앱처럼 사용 가능
- 세로 모드 권장
- 터치로 모든 기능 사용 가능
